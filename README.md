# 🔍 SEO 体检报告生成器

让不懂 SEO 的人也能看懂网站哪里需要改。

## 🎯 卖点

- **说人话**：不甩术语，每个问题都用"是什么→为什么重要→怎么改"讲清楚
- **改完涨多少分**：每项修复标注预计提升分数，改起来有动力
- **中文优先**：市面所有 SEO Skill 都是英文的，这是第一个中文的
- **分行业给建议**：自动识别电商/博客/SaaS/企业官网，给不同方案
- **同行对比**：输入竞品网址，看看差在哪

## 📦 安装

```bash
npx skills add YOUR_USERNAME/seo-audit
```

或直接复制 `SKILL.md` 到 `~/.claude/skills/seo-audit/`

## 🚀 使用

在 Claude Code 里说：
- "帮我看看这个网站的 SEO：https://xxx.com"
- "给 https://xxx.com 做个 SEO 体检"
- "seo audit https://xxx.com"

可选：提供竞品网址做对比
- "帮我分析 https://我的网站.com ，对比 https://竞争对手.com"

## 📊 检查维度（25+ 项，8 个维度）

1. 标题与摘要（15分）
2. 内容质量（15分）
3. 图片优化（10分）
4. 链接健康（10分）
5. 移动端友好（10分）
6. 社交传播（10分）
7. 技术基建（15分）
8. 代码质量（15分）

## 🌐 切换语言

默认输出中文。说 "output in English" 切换到英文报告。
